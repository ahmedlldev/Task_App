# MERN Task App

**Full Stack MERN Task App**

## Technology Used
- Mongo Database
- Express
- React
- Node.js

## Application Features
- Add New Tasks
- Delete Task
- Ordered List of Tasks
- Simple UI
- Good Color Theme
